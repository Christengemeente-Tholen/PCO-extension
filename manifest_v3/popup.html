Connect with the Planning Center

Fill in the generated Client ID and secret, in order to connect this extension to Planning Center. You can get both by requesting a Personal Access token from: https://api.planningcenteronline.com/oauth/applications This information is required to request things like the lyrics from Planning Center, because manifest V3 doesn't allow the browser to send request to the websites.
